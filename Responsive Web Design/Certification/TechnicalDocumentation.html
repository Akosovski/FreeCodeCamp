Virt Env


To Install Python virtual environment, you need to have a pip installed.


  • If you already have pip installed, type this in your terminal. It will install your env into your project folder.


    • pip install venv


  • Once the virtual environment has been installed, type this in your terminal according to your project's directory to activate the environment.


    • D:/*filedir/env/Scripts/Activate.ps1


    After that, your environment will be set as it is. Start building your django applications then.


    Install Apps


    To install a new application within the framework, we need to use the following command :



    python manage.py startproject *name*


  • After that, django will automatically add new files to the project according to the app name.

  • But you still need to add the app itself into the INSTALLED_APPS on settings.py file.


    • INSTALLED_APPS = ['yourappname',]


    Once the app is installed, you can proceed to modify them as you need.


    Set Database


    In this case, we use PostgreSQL to create a database and then connect it to the django.


    On the settings.py file, we set the parameters for the database as follows :



    DATABASES = {'default':{ENGINE, NAME, USER, PASSWORD, HOST, PORT}}


  • You need to fill the parameters of Engine, Name, User, Password, Host and Port in your settings.py file as shown.

    • Once you set them, the database will be automatically run after you relaunch the application.
