Installing Virtual Environment


To Install Python virtual environment, you need to have a pip installed.


  • Once pip has been installed, type this in your terminal. It will install your env into your project folder.


    • pip install venv


  • Once the virtual environment has been installed, type this in your terminal to activate the environment.


    • D:/django/env/Scripts/Activate.ps1


  • After that, your environment will be set as it is. Start building your django applications then.
